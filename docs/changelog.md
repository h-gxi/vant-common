# 更新日志

### 介绍

- 根据家校互动应用迭代开发更新，不定期发布。
- 大版本更新不完全兼容低版本，请先查阅更新日志。
- 小版本更新向下兼容，包含新特性和问题修复。
- beta属于测试版，新增组件和重构组件。

## 更新内容

### v0.4.0

`2024-10-10`

- 演示demo升级Vant 4版本。

### v0.3.2

`2024-10-08`

- 步骤条数字两边对齐。
- 表格显示边框时隐藏阴影部分显示。
- 修复表格底部边线隐藏显示问题。

### v0.2.9

`2024-09-18`

- 表格组件，增加nowrap属性，是否在一行内显示，超出内容显示省略号。
- 表格单元格内容居中问题。

### v0.2.6

`2024-09-11`

- 新增Popup组件，弹出层容器，可自定义扩展使用。
- 新增PopupRadio组件，弹出层单选项，继承自Popup组件。
- 新增FormCheckbox组件，表单多选项，可以配合Popup组件使用。

### v0.2.3

`2024-09-06`

- Table组件，新增cell-click事件，点击单元格时候触发。

### v0.2.2
