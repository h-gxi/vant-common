# 更新日志

### 介绍
- 根据家校互动应用迭代开发更新，不定期发布。
- 大版本更新不完全兼容低版本，请先查阅更新日志。
- 小版本更新向下兼容，包含新特性和问题修复。
- beta属于测试版，新增组件和重构组件。
